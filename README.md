# ❤️ The Josh & KJ Decision Machine

Mobile-first date and food jackpot roulette for GitHub Pages.

## Upload to GitHub Pages
Put these files at the root of your repository:

- `index.html`
- `manifest.webmanifest`
- `sw.js`
- `icon-192.png`
- `icon-512.png`
- optional `music/` folder with MP3 files

## Music
Create a folder named `music` and add any of these files if you want background music:

- `lofi.mp3`
- `happy.mp3`
- `arcade.mp3`
- `romantic.mp3`

Phones block autoplay with sound, so music starts after the first tap or when SPIN is pressed.

## Features
- Activities and Food modes
- Real slot-machine style reels
- Mood, budget, indoor/outdoor filters
- Challenges for each result
- Rare Love Jackpot events
- Spin history and stats
- Achievements and couple level
- Tiny music toggle and settings
- Day/night theme
- PWA install support
- Offline-ready service worker
